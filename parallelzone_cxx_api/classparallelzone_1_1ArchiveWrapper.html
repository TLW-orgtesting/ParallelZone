template<typename VariantArchive>
class parallelzone::ArchiveWrapper< VariantArchive >

ArchiveWrapper class enables serialization and deserialization of objects using various types of archives supported by MADNESS and Cereal.

This class wraps MADNESS/Cereal API for different output archive types. Required for serialization of Any.
