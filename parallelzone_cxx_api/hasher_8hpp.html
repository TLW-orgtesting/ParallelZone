◆ disable_if_hash_t

template<typename T >
using parallelzone::disable_if_hash_t = typedef std::enable_if_t<!is_hash_v<T> >

Disables an overload if T is the same as hash_type.

This type exploits SFINAE to disable templated overloads of functions when the user passes the hash instead of an object that needs hashing.
